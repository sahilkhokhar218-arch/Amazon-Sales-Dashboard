#  Amazon Sales Performance Dashboard

A Power BI dashboard designed to analyze Amazon product sales performance, pricing trends, discounts, customer ratings, and category-wise performance. This project provides actionable insights into product sales, customer engagement, and discount strategies using interactive visualizations.

---

##  Project Overview

The dashboard helps stakeholders understand:

* Product performance across different categories
* Impact of discounts on sales and ratings
* Category-wise revenue contribution
* Customer rating distribution
* Pricing analysis between actual and discounted prices
* Top-performing products and categories

---

##  Dataset Information

The dataset contains Amazon product information including:

| Column              | Description                    |
| ------------------- | ------------------------------ |
| Product ID          | Unique identifier for products |
| Product Name        | Product title                  |
| Category            | Product category               |
| Actual Price        | Original product price         |
| Discounted Price    | Selling price after discount   |
| Discount Percentage | Discount offered               |
| Rating              | Product rating                 |
| Rating Count        | Number of customer reviews     |

---

##  Dashboard Features

### KPI Cards

* Total Products
* Total Ratings
* Total Discount Percentage
* Total Actual Sales Value

### Interactive Filters

* Category Slicer
* Discounted Price Range
* Rating Count Filter

### Visualizations

#### 1. Category-wise Rating Analysis

* Compare ratings across product categories
* Identify high-performing categories

#### 2. Product Rating Distribution

* Donut chart showing rating contribution by products

#### 3. Discount Analysis

* Discount percentage comparison across categories
* Analyze discount effectiveness

#### 4. Sales Performance by Category

* Compare actual price and discounted price values
* Understand category revenue contribution

#### 5. Combined Performance Analysis

* Line and column chart showing:

  * Discount Percentage
  * Actual Price
  * Category Performance

##  Business Problem

Amazon offers products across multiple categories with varying prices, discounts, and customer ratings. Understanding how discounts influence customer behavior and category performance is essential for optimizing sales strategies.

This dashboard enables:

* Monitoring product performance
* Evaluating discount effectiveness
* Identifying top-performing categories
* Supporting data-driven business decisions
